# 股商测试 MBTI - 技术路线图

## 项目定位
一款面向年轻股民/基民的「投资人格测试」Web应用，融合MBTI框架与A股投资场景，
具有「高网感」文案风格和社交传播属性。

---

## Phase 1: 项目基础架构
- [x] Vite + React 项目脚手架
- [x] Tailwind CSS 样式框架
- [x] 组件结构设计
- [x] 路由设计（欢迎页 → 答题页 → 结果页）

## Phase 2: 题库构建（核心）
- [ ] 4维度（类比MBTI的E/I, S/N, T/F, J/P）
  - 维度1: 独立分析 ↔ 跟风从众
  - 维度2: 技术分析 ↔ 基本面分析
  - 维度3: 理性纪律 ↔ 情绪驱动
  - 维度4: 短线操作 ↔ 长期持有
- [ ] 每维度6题，共24题
- [ ] 5点李克特量表
- [ ] 每道题融入A股场景 + 互联网热梗

## Phase 3: 16型人格体系
- [ ] 16种投资人格原型定义
- [ ] 每种类型的创意命名（高网感）
- [ ] 投资风格描述、优势/劣势、适配策略
- [ ] 知名投资者对标

## Phase 4: 答题流程UI
- [ ] 欢迎页（品牌展示 + 开始按钮）
- [ ] 题目卡片动画（翻页/滑入效果）
- [ ] 进度条
- [ ] 选项交互动效

## Phase 5: 结果分析页
- [ ] 雷达图（4维度可视化）
- [ ] 人格类型结果卡
- [ ] 详细分析文案
- [ ] 分享功能（生成图片/复制链接）
- [ ] 「再测一次」入口

## Phase 6: 打磨 & 迭代
- [ ] 两Agent互审机制（构建者 ↔ 审查者）
- [ ] 文案网感打磨
- [ ] 移动端适配优化
- [ ] 动画细节调优
- [ ] SEO/社交Meta标签

---

## 技术栈
- React 18 + Vite
- Tailwind CSS
- recharts (雷达图)
- html2canvas (分享图生成)
- Framer Motion (动画)

## 部署方案
- GitHub Pages / Vercel 静态部署
